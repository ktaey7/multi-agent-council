# Grok CLI Adapter

Use Grok as a council participant when you want an independent counterargument,
web-aware research, or broad failure-mode analysis.

## Counterfactual Prompt

```text
Assume the current council consensus is wrong.

Question:
[question]

Current consensus:
[summary]

Provide:
1. Two realistic failure scenarios.
2. One reason the opposite decision may be correct.
3. One missed constraint.
4. Evidence that would change your view.
```

## Notes

- Keep web research separate from private repository context.
- If web access is used, cite sources in the final maintainer record.
- Do not send credentials, tokens, private customer data, or confidential logs.
